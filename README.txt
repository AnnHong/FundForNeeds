Installation manual for XAMMP

1. Download and install the latest version of XAMPP in your computer
2. The file is self-extracting archive, double click double click to run the extractor
3. A new folder named XAMPP is created containing Xampp control.exe and support folders and files
4. Click start Apache button. The name of module will become green color when it is running
5. Click start MySQL button. The name of module will become green color when it is running
6. Click on admin button to open the localhost webpage
